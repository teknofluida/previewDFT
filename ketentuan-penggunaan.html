Selamat datang di situs web PT. Dinamik Flow Teknologi. Situs web ini dimiliki dan dioperasikan langsung oleh PT. Dinamik Flow Teknologi (selanjutnya disebut sebagai “Dinamik Flow Teknologi”). Kami mohon Anda membaca syarat dan ketentuan penggunaan di bawah ini dengan seksama. Dengan mengakses atau menggunakan situs web ini, Anda menyetujui seluruh ketentuan penggunaan, kondisi, dan semua ketentuan hukum yang berlaku. Jika Anda tidak menyetujui ketentuan ini, maka kami mohon Anda untuk membatalkan penggunaan situs web ini.

1. Operasi Situs Web

Segala sesuatu yang dimuat dalam situs web ini dibuat dengan tujuan hanya untuk memberikan informasi umum dan Dinamik Flow Teknologi melakukan semua upaya yang wajar untuk memastikan bahwa informasi di situs ini benar pada saat dipublikasi. Akan tetapi kami tidak memberikan jaminan atau garansi mengenai keakuratan, ketepatan waktu, keabsahan dan keamanan dari informasi tersebut, sehingga penggunaan untuk kepentingan diluar tujuan yang kami maksudkan adalah menjadi tanggung jawab Anda.

Dinamik Flow Teknologi tidak dapat menerima permintaan pertanggung jawaban atas setiap kehilangan atau kerusakan, penggantian kerugian kepada pihak lain manapun juga yang disebabkan oleh penggunaan situs web ini, maupun yang disebabkan oleh adanya perubahan dan atau penghapusan informasi dan atau penangguhan atau penghentian situs web.

Dinamik Flow Teknologi berhak untuk mengubah atau menghapus setiap isi yang dimuat dan format dari situs web ini tanpa pemberitahuan sebelumnya. SahabatWaskita dapat menangguhkan atau menghentikan layanan dari situs web ini atas kebijakannya sendiri setiap saat tanpa pemberitahuan.

2. Hak Cipta dan Merek Dagang

Konten yang dimuat pada situs ini (informasi, grafis, video, audio, program, desain, dll) dilindungi oleh undang-undang hak cipta atau peraturan-peraturan yang terkait dalam bidang Kekayaan Intelektual (Intellectual Property Right). Setiap penggunaan konten demikian, tidak boleh direproduksi, diubah atau didistribusikan hanya dapat dilakukan dengan adanya ijin tertulis dari PT. Dinamik Flow Teknologi Indonesia. Materi dapat diunduh dari situs, disimpan pada komputer pribadi dan dicetak untuk kepentingan pribadi pelanggan atau untuk melakukan pemesanan kepada Dinamik Flow Teknologi.

"Dinamik Flow Teknologi" dan setiap nama produk atau layanan Dinamik Flow Teknologi adalah merek dagang atau merek dagang terdaftar dari PT. Dinamik Flow Teknologi. Merek dagang ini tidak boleh digunakan atau direproduksi tanpa ijin dari PT. Dinamik Flow Teknologi. Nama-nama produk dan nama-nama perusahaan lainnya yang disebutkan di situs web ini mungkin juga merupakan merek dagang atau merek dagang terdaftar dari pemiliknya masing-masing. Mohon untuk tidak menggunakan merek dagang tanpa izin dari pemegang hak.

3. Tautan ke Situs-Situs Web Pihak Ketiga

Situs web ini mungkin berisi tautan ke situs-situs web pihak ketiga yang tidak di bawah kendali atau dikelola oleh PT. Dinamik Flow Teknologi. Bahwa Dinamik Flow Teknologi menyediakan tautan-tautan ke situs-situs web pihak ketiga ini semata-mata dimaksudkan untuk kenyamanan Anda dan Dinamik Flow Teknologi tidak pernah bermaksud untuk merekomendasikan situs-situs dimaksud. Penggunaan segala bentuk informasi dari tautan dimaksud menjadi tanggung jawab Anda termasuk tanggung jawab kepada Pihak Ketiga. Dinamik Flow Teknologi tidak bertanggung jawab atas isi dari situs web pihak ketiga dan tidak bertanggungjawab atas segala kerusakan yang mungkin terjadi karena penggunaan tautan ke situs-situs web ini.

4. Perubahan pada Ketentuan Penggunaan

Dinamik Flow Teknologi dapat merevisi ketentuan penggunaan situs web ini setiap saat tanpa pemberitahuan terlebih dahulu dengan mengupdate posting ini, sehingga konfirmasi dari Anda setiap kali mengakses dan atau menggunakan situs ini diwajibkan

Untuk informasi mengenai bagaimana cara Dinamik Flow Teknologi menangani informasi pribadi pelanggan, silahkan melihat kebijakan privasi kami.
